## Introduction
인스타그램을 모티브로 해서 사용자들이 추억을 공유하는 Social Media 페이지를 구현했습니다.

## 사용한 기술
* Front-end: JavaScript | Next.js | emotion | Chakra-ui | Redux
* Back-end: Node | express | my-sql
* Others: AWS | ec2 | s3Bucket | AWS Lambda
